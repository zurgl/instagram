# Typst Instagram templates

## Editor

codium

## Dependencies

### Install rust

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### Install typst

```bash
cargo install typst-cli --locked
```

### Install a prettier

```bash
cargo install typstyle --locked
```

or

```bash
cargo install typstfmt --locked
```

### Install just

```bash
cargo install just --locked
```
